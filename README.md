# NODE version 14.7.0 

## Installation
npm i

## Storybook
npm run storybook

## Nuxt instance
npm run dev


## Themes location
src/assets/themes/

## global scss location
src/assets/library/

## build Theme (replace "white-label" with your theme name)
npm run build-theme white-label 

## Prime Theme View
first, rename config file to "vite.config.js" 
npm run prime-serve
